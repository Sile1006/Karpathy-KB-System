# Karpathy-KB-System
# 🚀 Karpathy-Style LLM 个人知识库系统

本项目是一个轻量级、无向量数据库的纯长上下文大语言模型知识库。专为全国大学生电子设计竞赛（NUEDC）备赛打造。

## 🌟 核心特性
- **纯原生长上下文**：摒弃传统 RAG 切片，100% 保留语义逻辑。
- **多模态摄入 (Ingest)**：支持直接解析 Markdown 笔记与 PDF 技术手册。
- **知识网状编译 (Compile)**：基于 LLM 自动提取技术实体，并生成支持 Obsidian 原生解析的双向链接 `[[ ]]` 图谱。
- **精准溯源问答 (Query)**：强制 LLM 附带信息来源，解决大模型幻觉问题。
- **自检优化 (Auto-Lint)**：利用正则表达式扫描死链，辅助知识网络闭环。
