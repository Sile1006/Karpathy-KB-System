# K230机器视觉部署与检测

## 模型训练与转换
* **算法选择**：主要使用 YOLOv8（及 YOLOv5）进行目标检测模型训练。
* **格式导出**：完成训练后，需将 `.pt` 权重文件成功导出为 ONNX 格式。
* **KModel 编译**：利用 nncase 工具链，解决算子兼容问题，将 ONNX 模型转换为适合 Canaan K230 部署的 `.kmodel` 格式。

## 视觉算法实现
* **形状识别**：使用 Python 编写视觉算法，专门用于检测视野中的矩形和菱形目标。
* **ROI测距系统**：通过识别外部参考框（Outer reference frames），结合已知的物理尺寸，反算并测量目标距离。
* **代码整合**：将自定义的测距算法与模型推理部分的代码进行修改与整合，确保在 K230 端侧高效、流畅地运行。
